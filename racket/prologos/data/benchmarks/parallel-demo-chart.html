Architecture

CALM Theorem: Monotone computations over fixed lattice structures converge to the same fixpoint regardless of evaluation order. This guarantees that parallel execution produces identical results to sequential.

BSP Scheduling: All propagators fire against the same immutable snapshot. Results are merged via commutative lattice joins. No locks, no coordination, no races.

Structural Capture: New cells and propagators created during fire are captured and deferred to the next round. The infrastructure handles all complexity — no module needs explicit parallelism awareness.

380/380 tests pass under true parallel execution. 10 cores. Racket 9 parallel threads. Zero correctness issues.

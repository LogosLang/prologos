Arithmetic Traits

Prologos provides generic arithmetic through six single-method traits: Add, Sub, Mul, Div, Neg, and Abs. Each trait defines one operation, and instances provide the implementation for each numeric type.

The parser keywords +, -, *, /, negate, and abs dispatch at the AST level based on the type of their arguments. These traits provide the same operations as first-class values that can be passed to higher-order functions like reduce.

Because each is a single-method trait, the dictionary IS the function — no wrapper struct needed.

Instance coverage: • Nat: Add, Sub (saturating), Mul — no Div, Neg, or Abs • Int: Add, Sub, Mul, Div, Neg, Abs • Rat: Add, Sub, Mul, Div, Neg, Abs • Posit8–64: Add, Sub, Mul, Div, Neg, Abs • String: Add (concatenation) — no others

Part I — Addition

prologos::core::add-trait

The Add Trait

Binary addition. Laws: commutative and associative.

require [prologos::data::nat :refer [add]]

trait Add {A}
  add : A A -> A
  :doc "Binary addition over type A"
  :laws
    - :name "commutative"
      :forall {x y : A}
      :holds [eq? [add x y] [add y x]]
    - :name "associative"
      :forall {x y z : A}
      :holds [eq? [add [add x y] z] [add x [add y z]]]

Nat instance — delegates to prologos::data::nat/add

impl Add Nat
  defn add [x y] <Nat>
    add x y

Add String — Concatenation

Part V — Negation and Absolute Value

prologos::core::neg-trait

The Neg Trait

Unary negation. No Nat instance (negation leaves the Nat domain). Law: involution (neg (neg x) = x).

trait Neg {A}
  neg : A -> A
  :doc "Unary negation over type A"
  :laws
    - :name "involution"
      :forall {x : A}
      :holds [eq? [neg [neg x]] x]

Int instance

impl Neg Int
  defn neg [x : Int] <Int>
    int-neg x

prologos::core::abs-trait

The Abs Trait

Absolute value. No Nat instance (Nat is always non-negative). Laws: non-negative and idempotent.

trait Abs {A}
  abs : A -> A
  :doc "Absolute value over type A"
  :laws
    - :name "non-negative"
      :forall {x : A}
      :holds [gte? [abs x] 0]
    - :name "idempotent"
      :forall {x : A}
      :holds [eq? [abs [abs x]] [abs x]]

Int instance

impl Abs Int
  defn abs [x : Int] <Int>
    int-abs x

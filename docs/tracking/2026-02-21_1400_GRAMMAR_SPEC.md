## Ott Compilation Fixes (2026-02-21)

Fixed multiple Ott 0.34 compilation errors:
- **Self-loop error**: Renamed `m0`/`m1`/`mw` → `Zero`/`One`/`Omega` (production names looked like metavar+index)
- **Metavar conflicts**: Changed mult alias from `m` to `q`; removed `typevar` and `nat` metavars
- **Invalid metavar names**: `body`/`arg` → `e1`/`e2`; `S'` → `S1`/`S2`; `q'` → `q`
- **Invalid metaproductions**: Removed `zeros(i)` and `single(i,j)` from usage context
- **Parse ambiguity**: Link rule `c1:S1, c2:S2` → `c1:S1, c2:S2, emptyC` to disambiguate from branch syntax
- **LaTeX wrapper**: Updated to use correct Ott-generated commands (`\ottdefn*` singular, not `\ottdefns*` plural)
- **Result**: 47 definition rules, 81 clauses, 0 errors

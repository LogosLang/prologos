### Decisions & Lessons Learned

1. **Session-name-as-implicit-label**: Anonymous `(Mu body)` inside `(session Loop ...)` uses `'Loop` as the recursion label. This makes the common pattern `(session Loop (Mu (Send Nat (SVar Loop))))` work naturally ‚Äî no need for `(Mu Loop (Send Nat (SVar Loop)))` explicit label syntax.

2. **Built-in type resolution**: The parser converts `Nat` ‚Üí `(surf-nat-type)` ‚Üí `(expr-Nat)` (canonical AST). This means session types elaborate to canonical forms, not `(expr-fvar 'Nat)`, which is critical for `check` patterns in `typing-core.rkt` to match.

3. **Nat vs Int literal**: `42` in sexp mode is `(surf-int-lit 42)` ‚Üí `(expr-int 42)` (type: Int). `42N` is `(surf-nat-lit 42)` ‚Üí `(expr-nat-val 42)` (type: Nat). Session types must match their literal types.

4. **Unannotated proc-recv**: Process `(proc-recv self x ...)` doesn't annotate x's type ‚Äî the type should come from the session context. Using `#f` as annotation and adding `(or (not a-annot) ...)` guard in typing-sessions is the correct fix. The `is-type` check for the session's receive type still works because `(expr-Nat)` is a canonical type that `infer` returns `(expr-Type (lzero))` for, even with empty gamma.

5. **S3c `throws` deferred**: The `throws` desugaring needs capability integration (S5), so it's deferred until capabilities are built.

### Additional Decisions & Lessons Learned

6. **Monotonic lattice and intermediate cells**: Propagator cells never decrease ‚Äî intermediate cells retain their session values (e.g., `sess-send(String, End)`) after decomposition. The deadlock/completeness check must only flag "terminal" cells (those with 'stop operations), not intermediate decomposition cells.

7. **Branch continuation cells need context**: When `proc-case`/offer creates branch continuation cells, they need trace entries too. Without them, contradictions at branch cells have empty derivation chains. Solution: record "branch 'X of offer on Y" on each branch cell.

8. **`srcloc-unknown` for core AST**: Core `proc-*` AST nodes don't carry source locations. Session errors use `srcloc-unknown` from `source-location.rkt`. Surface forms (parsed from source) will carry locations once the surface‚Üîcore bridge is tightened.

9. **Struct inheritance and match ordering**: `session-protocol-error` extends `session-error`. In `format-error`, the more specific `session-protocol-error` pattern must come BEFORE the generic `session-error` pattern, or the generic clause matches first.

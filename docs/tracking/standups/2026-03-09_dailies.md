## Key Technical Insights

- **Performance regression pattern**: Adding new pattern kinds (`'int-lit`) to pattern compilation required updating `pattern-is-simple-flat?` to recognize `'wildcard` as well as `'var` — missing this caused the fast path to fall through to the slow `compile-match-tree` path for ALL patterns (850s → 186s regression).
- **Test output format drift**: `count-answers` regex `\{map` failed silently when narrowing output changed to `{:key value}` format. Tests returned 0 answers instead of failing — a reminder that "passes with 0" is a dangerous test pattern.
- **Whale file splitting**: Fixture compilation (shared `define-values`) dominates test time — splitting below ~19 tests per file yields diminishing returns since both halves pay the same fixture cost.
- **Concurrent agent edits**: 10 parallel split agents editing `dep-graph.rkt` caused "file modified since read" errors. Sequential dep-graph updates would be safer for future splits.

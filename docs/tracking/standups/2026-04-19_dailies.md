- *Structurally emergent*: the classification flows Tier 1 → Tier 2 → Tier 3 by inheritance; enforcement reads it at propagator install. ✓
- *Information flow*: domain → classification → enforcement decision; all via registered structures. ✓
- *ON-NETWORK* ⚠️: **Tier 2 registry lives as a Racket parameter (not a cell) in 4C** — `register-merge-fn!/lattice` mutates a registry hash.
